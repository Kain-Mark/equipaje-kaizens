## 📌 Sobre el proyecto
**Equipaje Kaizens** es una Landing Page y web de nicho orientada al marketing de afiliación (Amazon Afiliados). El proyecto nace con el objetivo de resolver el problema del "spam visual" común en las webs de afiliados, apostando por un diseño editorial, limpio y minimalista.

El diseño se basa en la filosofía japonesa *Kaizen* (mejora continua), enfocada en el orden y la selección curada de productos (mochilas, maletas y accesorios) para viajeros y profesionales.

## 🛠️ Tecnologías utilizadas
Este proyecto está construido desde cero sin el uso de frameworks pesados, primando el rendimiento y el control absoluto del código:
* **HTML5:** Estructura semántica completa (`<section>`, `<article>`, `<nav>`, etc.) para maximizar el SEO y la accesibilidad (atributos `aria-label`).
* **CSS3:** Diseño responsivo *Mobile-First*. Uso intensivo de CSS Variables (Custom Properties), CSS Grid y Flexbox para layouts estructurados.
* **JavaScript (Vanilla):** Interactividad ligera. Gestión de menú móvil, *smooth scrolling* para navegación interna, *Intersection Observer* para resaltar elementos activos en la cabecera y *Lazy Loading* nativo para optimización de imágenes.

## 🚀 Características destacadas
- **UX/UI Editorial:** Diseño inspirado en revistas de estilo de vida, abandonando la típica estética de "catálogo" para generar mayor confianza en el usuario.
- **Explorador de marcas tipográfico:** Uso estratégico de Google Fonts (`Playfair Display`, `Oswald`, `Barlow`) para diferenciar las marcas visualmente de forma elegante y rápida (sin carga excesiva de imágenes de logos).
- **Estrategia de Conversión:** Botones de "Llamada a la Acción" (CTA) estratégicamente ubicados y destacados, integración de componentes "Producto del mes" y estructuración de contenido pensado para el programa de afiliados.
- **Rendimiento Optimizado:** Código ligero, imágenes con atributos de carga diferida y sin dependencias externas que ralenticen la web.
